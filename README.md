# FF4 Battle Auto Bot

## 概要
Final Fantasy IV の戦闘を自動化するBotです。
画面上のUIを画像認識し、戦闘開始判定、攻撃、HP低下時の回復、勝利後の進行を自動で行います。

## 背景・目的
ゲームの操作自動化や、状態に応じたUI判定の仕組みに興味があり、
画面認識を用いたBot開発に取り組みました。
将来的には、単純な操作自動化だけでなく、異常検知やより高度な自動QAに発展させたいと考えています。

## 実装している機能
- 左右移動による敵エンカウント
- セシルUIやATBフレームの検出による戦闘開始判定
- 「たたかう」ボタン検出後の自動攻撃
- セシルのHP低下時のポーション使用
- 勝利画面検出後の進行操作
- ESCキーによる強制停止

## 工夫した点
- 画像認識失敗時に処理が落ちにくいよう例外処理を実装
- 画面遷移直後の揺れを考慮し、攻撃ボタンを複数回確認する待機処理を実装
- 回復対象やアイテムが見つからない場合は、メニューを戻して処理を安全に終了するよう設計

## 今後の改善
- バックアタックなど通常戦闘と異なるUI配置を、異常状態として検知・停止する機能
- 異常発生時のスクリーンショット保存
- ログ出力の整理
- Jenkins等を用いた自動実行への拡張
